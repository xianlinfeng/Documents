＃　编辑代码
Ctrl + Tab: 切换文件
Ctrl + [ or Ctrl + ]: 代码缩进
Shift + Alt + F: 自动格式化代码
Ctrl + Enter：在当前行下边插入一行
Ctrl + Shift+ Enter: 在当前行上边插入一行
End: 光标移动到行尾
Home: 光标移动到行首
Ctrl + Shift + L: 同时选中所有匹配
Ctrl + Shift + K: 删除行
Ctrl + . 快速解决
Ctrl + ' 显示集成终端
Alt + Shift + A: 多行注释



＃　主命令框
Ctrl + Shift + P: 打开命令面板
